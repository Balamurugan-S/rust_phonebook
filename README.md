# rust_phonebook
A simple Phonebook application using RUST 
(A part of my Rust Practice Projects)

*---------------------*
My RUSTY Phonebook
*---------------------*
1. List   Contacts
2. Add    Contacts
3. Delete Contacts
4. Find   Contacts
5. EXIT
*---------------------*

User can list add delete and find contacts

Entries are stored as vectors in a struct ([name:String, number:String])
